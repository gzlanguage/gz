Installation Guide

1. On iPhone

In Safari:

  1. Tap the Share icon (a square with an arrow pointing up).
  2. Select Add to Home Screen.
  3. Confirm by tapping Add.

2. On Android

In Chrome:

  1. You may see a prompt at the bottom asking if you want to install the site as an app. Tap Install.
  2. If the prompt does not appear:
    • Open the Menu (three dots in the top-right corner).
    • Select Add to Home Screen.
    • Confirm by tapping Add.
